## 🏗️ Platform Architecture

### URL Structure

```
pumthai.com/
│
├── /                             # Landing Page
├── /login                        # Login
├── /register                     # Register
│
├── /seller/                      # 🏪 Seller Center
│   ├── /                         # Dashboard
│   ├── /store                    # ตั้งค่าร้าน
│   ├── /services                 # จัดการบริการ
│   ├── /orders                   # ออเดอร์ทั้งหมด
│   ├── /orders/[id]              # รายละเอียดออเดอร์
│   ├── /clients                  # ลูกค้า
│   ├── /team                     # 👥 ทีมงาน
│   │   ├── /                     # Dashboard ทีม
│   │   ├── /members              # สมาชิกทีม
│   │   ├── /jobs                 # งานทั้งหมด
│   │   ├── /jobs/new             # สร้างงาน
│   │   ├── /jobs/[id]            # รายละเอียดงาน
│   │   └── /payouts              # จ่ายเงินลูกทีม
│   ├── /accounts                 # 🏪 บัญชีที่ลงขาย
│   ├── /accounts/new             # ลงขายบัญชีใหม่
│   ├── /promotions               # โปรโมชั่น
│   ├── /analytics                # สถิติ
│   └── /settings                 # ตั้งค่า
│
├── /work/                        # 👤 Worker App
│   ├── /                         # งานที่รับได้
│   ├── /my-jobs                  # งานของฉัน
│   ├── /my-jobs/[id]             # รายละเอียดงาน
│   ├── /earnings                 # รายได้
│   ├── /withdraw                 # ถอนเงิน
│   └── /profile                  # โปรไฟล์
│
├── /market/                      # 🏪 Account Marketplace
│   ├── /                         # หน้าแรก Marketplace
│   ├── /[category]               # หมวดหมู่ (facebook, instagram, etc.)
│   ├── /listing/[id]             # รายละเอียดบัญชี
│   └── /my-orders                # ออเดอร์ของฉัน
│
└── /s/[slug]/                    # 🏪 Public Store (หน้าร้าน)
    ├── /                         # หน้าร้าน
    ├── /[serviceId]              # รายละเอียดบริการ
    ├── /order                    # สั่งซื้อ
    ├── /status                   # ดูสถานะ
    └── /reviews                  # รีวิว
```

## 👤 User Roles

### 1. 🏪 Seller (ผู้ขาย)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                              │
│  🏪 Seller                                                                  │
│                                                                              │

│  สามารถ:                                                                    │
│  ├── 🏪 มี Store (หน้าร้าน)                                                 │

│  ├── 🏪 ขายบัญชี Social Media                                              │

│                                                                              │

```

#### ขั้นตอนการยืนยัน:

1. **Worker ลงทะเบียนบัญชี Social**
   - เลือก Platform (Facebook, Instagram, TikTok, etc.)
   - กรอก Username / Profile URL
   - อัพโหลด Screenshot หน้าโปรไฟล์

2. **AI ตรวจสอบอัตโนมัติ** (OpenAI Vision)
   - ✅ มีรูปโปรไฟล์
   - ✅ จำนวน Followers ตามเกณฑ์
   - ✅ ไม่ใช่หน้า Login / Error
   - ✅ Username ตรงกับที่กรอก

3. **ผลลัพธ์**
   - ✅ **ผ่าน** → บัญชีพร้อมใช้งาน
   - ⚠️ **ไม่แน่ใจ** → ส่ง Admin ตรวจ Manual
   - ❌ **ไม่ผ่าน** → แจ้งเหตุผล + ให้ลองใหม่

#### สิ่งที่ AI ตรวจสอบ:

| รายการ | ตัวอย่าง |
|--------|---------|
| 📷 รูปโปรไฟล์ | ต้องมีรูป (ไม่ใช่รูป default) |
| 👤 ชื่อบัญชี | ตรงกับที่กรอก |
| 👥 Followers | ≥ 50 (ขั้นต่ำของระบบ) |
| 📅 ความเก่า | ดูจาก UI (ถ้าเห็น) |
| 🚫 ปลอม | ตรวจ Screenshot ปลอม |
