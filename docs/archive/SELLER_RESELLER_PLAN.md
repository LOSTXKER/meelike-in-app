### 2. 👥 Client Management

**วัตถุประสงค์:** จัดการข้อมูลลูกค้าของ Reseller

```typescript
// Types
interface ResellerClient {
  id: string;
  sellerId: string;
  
  // Basic Info
  name: string;
  nickname?: string;
  contactPerson?: string;
  
  // Contact
  phone?: string;
  email?: string;
  socialMedia?: {
    facebook?: string;
    line?: string;
    instagram?: string;
  };
  
  // Organization
  tags: string[];
  notes?: string;
  
  // Stats
  totalSpent: number;
  totalOrders: number;
  totalProfit: number;
  
  // Pricing
  customPriceMultiplier?: number;  // 1.0 = ราคาปกติ, 1.2 = +20%
  
  // Timestamps
  createdAt: string;
  lastOrderAt?: string;
}

interface ClientTag {
  id: string;
  sellerId: string;
  name: string;
  color: string;
  clientCount: number;
}
```

## 🎯 สรุป

**Reseller (Agent) System** คือฟีเจอร์เสริมใน MeeLike Seller ที่ช่วยให้ Seller สามารถ:

1. **จัดการลูกค้า** - เก็บข้อมูลลูกค้า แบ่งกลุ่มด้วย Tags
2. **สั่งออเดอร์ในนามลูกค้า** - เลือกลูกค้า เลือกบริการ ระบุราคาขาย
3. **คำนวณกำไรอัตโนมัติ** - เห็นต้นทุน ราคาขาย กำไร แบบ Real-time
4. **วิเคราะห์ผลประกอบการ** - รายงานกำไรแยกตามลูกค้า/บริการ/ช่วงเวลา
5. **รับออเดอร์จาก Store** - ออเดอร์จากหน้าร้านบันทึกเป็นออเดอร์ Reseller

ทั้งหมดนี้ใช้ **Design System** และ **Subscription System** ร่วมกับ MeeLike Seller ทำให้ไม่ต้องสร้างระบบใหม่ทั้งหมด
